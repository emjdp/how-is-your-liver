# TONE_GUIDE — 톤 가이드

## 브랜드 보이스
- **차분한 풍자가.** 헬스 리포트의 톤으로 블랙코미디를 던진다.
- 2인칭 "당신"을 사용, 명령조 지양.
- 1~2문장으로 끝낸다. 길게 늘이지 않는다.

## 카피 원칙
1. 자랑은 가능 — 칭찬은 안 한다.
2. 숫자가 주인공, 카피는 1~2문장.
3. 항상 자기 객관화 1스푼.
4. 마지막에는 부드럽게 경고.

## 허용 표현 (예시 풀)
- "기록은 화려하지만, 간은 조용히 울고 있습니다."
- "스토리각은 나왔고, 내일 컨디션은 비공개입니다."
- "오늘의 간은 비상근무 체제에 들어갔습니다."
- "공유는 가능한 기록입니다. 건강에는 비추천입니다."
- "술자리는 끝났지만, 간의 야근은 이제 시작입니다."
- "최근 7일 출석률은 성실한데, 방향이 조금 잘못됐습니다."
- "내일의 너는 오늘의 너를 고소하고 싶을 수 있습니다."
- "오늘 간의 잔업수당은 0원입니다."
- "알림: 간이 야근 신청 없이 야근 중입니다."
- "이 기록은 '잘했다'가 아니라 '괜찮냐'로 읽어 주세요."

## 금지 표현 (자동 테스트 — `tests/safety-copy.test.ts`로 검사)
- "더 마셔", "한 병 더", "계속 달려", "달리자"
- "주신", "술고수", "주당", "주신급", "레전드", "전설"
- "진정한 상남자", "찐 남자", "여자가 이 정도면"
- "아직 부족", "이 정도면 더", "더 가자", "한 잔만 더"
- "음주운전", "대리 없이"
- "치료", "진단", "처방" (의학적 단정)
- "축하드립니다", "축하합니다" — 과음 결과 축하로 읽힘
- "자랑하기엔 부족", "자랑하기 부족" — 더 마시면 자랑 가능 뉘앙스

> **검사 범위**: `data/tierMessages.ts`의 모든 4슬롯 카피 + 버튼 라벨 + 빈 상태/토스트 카피 + **v1.0 신규 카드 템플릿 카피 (tpl_forecast, tpl_warning, tpl_weekly)**.
> **예외**: `lib/safety-copy.ts`의 고정 안전 안내 1줄은 "운전" 단어를 포함한다(안전 목적). 이 한 줄은 금지어 검사에서 제외(allow-list).
> **참고**: 단어 "운전"은 일반 금지어에서 빠졌다 — 안전 안내가 정상 통과하도록. 의심스러우면 "음주운전" / "대리 없이" 패턴으로만 걸러낸다.
> **중요**: v1.0에서 추가하는 모든 카드 템플릿 카피도 동일한 금지어 검사 대상이다.

## v1.0 신규 안내 문구 예시

### 하이볼 입력 안내
- "하이볼은 일반 레시피 기준 추정입니다." (입력 카드 내 보조 텍스트)
- 톤: 짧고 중립적. 불안감 조성 금지. 정보 제공 목적.

### 사용자 사진 배경 카드 안내
- "사진은 기기 안에서만 사용되고 서버로 전송되지 않아요." (PhotoPicker UI 내 안내)
- 톤: 안심을 주는 1줄. "절대" 같은 과강조 어 금지. 사실을 단순하게 전달.

## 위험한 표현 (회피)
- "상위 N%" — 실제 통계 부재.
- "다음 티어까지 N병" — 게임화 → 과음 유도.
- "친구보다 잘 마심" — 경쟁.
- "어른" / "성인 인증 완료" — 자기효능 보상.
- "성장", "다음 단계", "숙련" — 음주량 증가를 보상으로 미화 (소맥 견습생 카피 주변에서 특히 주의).
- "자랑" 단어의 반복 — 브랜드 핵심이지만 페이지당 1회 정도로 제한.

## 버튼 문구
| 의도 | 권장 | 비권장 |
|---|---|---|
| 결과 보기 | "이 날의 간 상태 보기" | "오늘 점수 확인" |
| 카드 만들기 | "스토리 카드 만들기" | "자랑하기" |
| 저장 | "이미지 저장" | "전설 박제" |
| 수정 | "기록 수정" | "기록 갱신" |
| 주간 보기 | "최근 7일 간 리포트" | "주간 랭킹" |

## 결과 문구 (티어별, 4슬롯)
티어 카피는 다음 4슬롯을 가진다. ([TIER_SYSTEM](./TIER_SYSTEM.md)과 1:1 매핑.)
- `headline` — 결과 페이지 상단, ≤24자.
- `subline` — 보조 1줄, ≤40자.
- `cardLine` — 스토리 카드용, ≤18자.
- `safetyLine` — 반드시 1줄 경고/현타, ≤40자.

## 스토리 카드 문구
- `cardLine` ≤18자.
- 보조 수치 라벨 ≤8자.
- 안전 안내 ≤40자 (한 줄 유지).

## 안전 안내 문구 (고정)
모든 결과·카드 하단에 1줄 고정:
> "이 기록은 엔터테인먼트/참고용 추정치입니다. 일반 성인 기준이며, 개인차가 큽니다. 음주 후 운전은 절대 금지."

## 자랑과 자학 사이의 균형 규칙
- **낮은 티어**: 자랑 톤 0% / 자학 톤 30% / 안내 톤 70%.
- **중간 티어**: 자랑 30 / 자학 50 / 안내 20.
- **높은 티어**: 자랑 10 / 자학 70 / 경고 20 (높은 티어일수록 멋있게 그리지 않는다).

## 금지 방향 (총괄)
- 과음 찬양 / 폭음 경쟁 유도.
- 성별 고정관념, 남성성·여성성 기반 술부심.
- 미성년자 음주 암시.
- 음주운전 관련 가벼운 농담.
- 의학적 진단처럼 보이는 표현.
- 정확한 건강 판정처럼 오해될 표현.
