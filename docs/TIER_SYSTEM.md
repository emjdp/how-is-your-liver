# TIER_SYSTEM — 티어 시스템

## 설계 원칙
- 당일/주간 티어를 **분리**한다.
- 각 7단계. 높을수록 **자학·경고** 강도↑.
- "다음 티어까지 N병" 같은 게임화 표현 금지.
- 랭킹·"상위 N%" 표현 금지.

## 당일 티어 (기준: 그날 순수 알코올 g)
| ID | 이름 | 조건 (g) | 톤 | 색 |
|---|---|---|---|---|
| `t0_peace` | 간 평화주의자 | `g == 0` | 휴식 | `--color-lime` |
| `t1_light` | 가벼운 주객 | `0 < g <= 20` | 가벼움 | `--color-deep-green-2` |
| `t2_somaek` | 소맥 견습생 | `20 < g <= 45` | 익숙 | `--color-deep-green-2` |
| `t3_yajang` | 야장 정규직 | `45 < g <= 80` | 자랑 + 자학 시작 | `--color-deep-green` |
| `t4_overtime` | 간 야근조 | `80 < g <= 120` | 자학 본격 | `--color-deep-green` |
| `t5_overload` | 간혹사범 | `120 < g <= 180` | 경고 톤 | `--color-warn` (소량) |
| `t6_redzone` | 간 과부하 경보 | `g > 180` | 강한 자학 + 경고 | `--color-warn` |

> 참고 환산: 소주 1병 ≈ 46.87g, 맥주 1잔 ≈ 17.75g.
> 경계는 모두 `<=` / `>` 기준 (예: 20g 정확히는 `t1_light`, 20.01g는 `t2_somaek`).

## 주간 티어 (기준: 최근 7일 합산 순수 알코올 g)
| ID | 이름 | 조건 (g) |
|---|---|---|
| `w0_rest` | 간 안식주 | `g == 0` |
| `w1_breeze` | 산들주간 | `0 < g <= 80` |
| `w2_routine` | 주중 적당 | `80 < g <= 200` |
| `w3_busy` | 간 출근주간 | `200 < g <= 400` |
| `w4_overtime` | 간 야근주간 | `400 < g <= 700` |
| `w5_overload` | 간 비상주간 | `700 < g <= 1000` |
| `w6_redzone` | 간이 사표 쓴 주 | `g > 1000` |

## 티어별 카피 (4슬롯, 모든 티어에 필수)

각 티어는 다음 4슬롯을 가진다. ([TONE_GUIDE](./TONE_GUIDE.md) 참조.)
- `headline` ≤24자
- `subline` ≤40자
- `cardLine` ≤18자
- `safetyLine` ≤40자

## 색상/분위기
- **낮은 티어**: 라임 + 딥그린, 밝은 톤.
- **중간 티어**: 딥그린 위주.
- **높은 티어**: 딥그린 + 좁은 면적의 warn 컬러 (테두리·아이콘만, 배경은 안 함 — 자극 회피).

## 과음 경쟁 방지 장치
- "다음 티어까지" 표시 금지.
- 티어를 가로로 늘어놓고 현재 위치를 강조하는 시각화 금지.
- 카드에 "랭킹" 단어 사용 금지 — 항상 "오늘의 상태"로 표현.
- 높은 티어일수록 카드의 채도·밝기를 낮춰 시각적 보상감을 줄인다.
